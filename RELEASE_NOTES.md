# StedNet-Lock Release Notes

## Version 1.0.0

This is the initial release of StedNet-Lock.

### What's New

* **Shizuku Integration**: Switch network modes using secure system APIs without root.
* **Network Mode Locking**: Toggle and lock advanced network modes (e.g. LTE/NR/WCDMA).
* **Profiles**: Save and quickly restore customized network configurations.
* **Closed Source release**: Production build package optimized for speed and stability.

### Installation Instructions

* Download the `StedNet-Lock-v1.0.0.apk` from the assets section below.
* Ensure Shizuku is running on your device, then launch the app and grant permission.
